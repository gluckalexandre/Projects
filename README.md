# Projects

## 1. Tractus Asia – Prometheus Infrastructure Analysis
**Tools:** Microsoft Excel, ArcGIS, Microsoft PowerPoint

During my internship at Tractus Asia, I worked on the Prometheus project analyzing industrial infrastructure across Thailand.

The project involved:
- Preparing and structuring infrastructure datasets in Excel
- Mapping infrastructure locations using ArcGIS
- Visualizing geographic clusters of industrial facilities
- Supporting strategic insights for infrastructure planning

Key output included geographic visualization of infrastructure clusters across provinces such as Rayong, Chon Buri, and Samut Prakan.

## 2. Fuel Inventory Data Analysis
**Tools:** Python (Pandas, Matplotlib, Seaborn)

This project analyzes fuel inventory data to understand patterns in fuel levels, refill frequency, and operational efficiency.

- Exploratory data analysis (EDA)

- Identifying trends in fuel consumption

Project files include the full Python analysis and report.
